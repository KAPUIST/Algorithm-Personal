## 문제 설명

두 정수 a, b가 주어졌을 때 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수, solution을 완성하세요.
예를 들어 a = 3, b = 5인 경우, 3 + 4 + 5 = 12이므로 12를 리턴합니다.

## 문제풀이

```
function solution(num) {

  if(num === 1){
      return 0
  }
    for(let i = 0; i < 500; i++){
        if(num === 1) return i
        if(num % 2 === 0) {
            num = num / 2;
        }
        else if(num % 2 === 1) {
            num = (num * 3) +1
        }
    }
    return -1
}
```
